# Тестовое задание

Стек:

1. React typescript
2. Redux, redux-thunk
3. Unit test - Jest
4. E2E test - Cypress
5. UI - AntDesign

---

Для полноценной работы необходимо запустить backend на порте :8080

В корне проекта выполнить 
### `npm install`

### `npm start`
Запуск
[http://localhost:3000](http://localhost:3000)


### `npm test`

Запуска тестов jest

### `npm run cypress`

Запуск тестов cypress
